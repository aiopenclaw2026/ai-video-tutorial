## 📝 教程内容大纲

### 步骤 1: 准备工作
- 注册账号（可灵、即梦、Runway 等）
- 了解基本概念（文生视频、图生视频）
- 准备素材（文案、图片、参考视频）

### 步骤 2: 文案创作
- 使用 AI 写脚本（ChatGPT、Claude 等）
- 分镜设计
- 时长规划

### 步骤 3: 图像生成
- 使用 AI 生成图片（Midjourney、Stable Diffusion、即梦等）
- 图片风格统一
- 尺寸和格式要求

### 步骤 4: 视频生成
- 图生视频工具使用（可灵、即梦、Runway、Pika）
- 参数设置（时长、运动强度、种子等）
- 多次尝试和优化

### 步骤 5: 视频编辑
- 剪辑软件选择（剪映、CapCut、Premiere）
- 拼接视频片段
- 转场效果

### 步骤 6: 配音和音效
- AI 配音工具（ElevenLabs、剪映配音）
- 背景音乐选择
- 音效添加

### 步骤 7: 字幕和特效
- 自动字幕生成
- 字幕样式调整
- 特效添加

### 步骤 8: 导出和发布
- 导出参数设置
- 平台要求（抖音、B 站、YouTube）
- 封面设计

## 🔧 技术方案

### 技术选型

| 功能 | 技术 | 说明 |
|------|------|------|
| 页面结构 | HTML5 | 语义化标签 |
| 样式 | CSS3 | Flexbox/Grid 布局 |
| 交互 | Vanilla JS | 无需框架，轻量快速 |
| 数据存储 | localStorage | 保存用户进度 |
| 图标 | SVG / Emoji | 无需外部依赖 |
| 部署 | 静态托管 | GitHub Pages / Vercel |

### 无需外部依赖

- 不使用 jQuery、Vue、React 等框架
- 不使用 Bootstrap 等 CSS 框架
- 不依赖 CDN 资源（可选）
- 单文件即可运行（可选合并版本）
