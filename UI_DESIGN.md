## ✨ 动画效果

### 步骤切换
```css
/* 内容淡入 */
fade-in: 0.3s ease-out

/* 卡片滑入 */
slide-in: 0.3s cubic-bezier(0.4, 0, 0.2, 1)
```

### 按钮交互
```css
/* 悬停 */
transform: translateY(-1px)
transition: 0.2s ease

/* 按下 */
transform: translateY(0)
```

### 进度条
```css
/* 进度增长 */
width: 0% → 62.5%
transition: width 0.5s ease
```

## 🎯 交互细节

### 侧边栏步骤点击
1. 点击步骤卡片
2. 卡片背景变为悬停色
3. 主内容区平滑滚动到对应步骤
4. 进度条更新
5. 本地存储保存进度

### 下一步/上一步
1. 按钮悬停时有位移反馈
2. 点击后内容区切换
3. 新内容淡入动画
4. 自动滚动到顶部

### 搜索功能
1. 点击搜索图标展开搜索框
2. 输入时实时过滤步骤
3. 高亮匹配关键词
4. ESC 关闭搜索
